# Augustinović - Kvalitetni Sustavi za Filtraciju Vode

Profesionalna rješenja za filtraciju i omekšavanje vode. Ovaj projekt je moderna Next.js aplikacija koja služi kao landing stranica za **Augustinović** usluge.

## 🚀 Tehnologije

- **Next.js 16.1.6** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **Montserrat Font** (Premium tipografija)
- **Framer Motion** (Glatke animacije)
- **Lucide React** (Moderne ikone)
- **React PDF** (Interaktivni pregled specifikacija)

## 🛠️ Instalacija i pokretanje

1. Klonirajte repozitorij
2. Instalirajte ovisnosti:
   ```bash
   npm install
   ```
3. Pokrenite razvojni poslužitelj:
   ```bash
   npm run dev
   ```
4. Izgradite produkcijsku verziju:
   ```bash
   npm run build
   ```

## 📂 Struktura projekta

- `src/components/` - Višekratne React komponente (Navbar, Hero, ROSection, etc.)
- `src/app/` - Next.js App Router stranice i layouti.
- `public/assets/` - Slike, video materijali i PDF dokumentacija.
